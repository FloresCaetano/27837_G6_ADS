## Historias de Usuario Prioritarias

### Cliente:
- **HU-001:** Como cliente, quiero diseñar una mezcla personalizada seleccionando ingredientes, para crear un producto unico.
- **HU-002:** Como cliente, quiero ver el valor nutricional total de mi mezcla, para tomar decisiones informadas.
- **HU-003:** Como cliente, quiero realizar un pedido y consultar su estado, para hacer seguimiento.

### Propietario:
- **HU-010:** Como propietario, quiero gestionar el inventario de productos, para mantener control del stock.
- **HU-011:** Como propietario, quiero cambiar el estado de los pedidos, para organizar la preparacion y entrega.
- **HU-012:** Como propietario, quiero ver reportes de ventas, para analizar el desempeño del negocio.
