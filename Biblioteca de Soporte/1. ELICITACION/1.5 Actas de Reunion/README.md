## Proposito

Registrar formalmente las decisiones, acuerdos y compromisos alcanzados durante las sesiones de trabajo con el cliente y el equipo de desarrollo.

## Estructura de una Acta

1. **Fecha y hora:** Cuando se realizo la reunion
2. **Participantes:** Asistentes (cliente, equipo, otros stakeholders)
3. **Agenda:** Temas a tratar
4. **Desarrollo:** Discusion y puntos clave
5. **Acuerdos y decisiones:** Compromisos asumidos
6. **Proximos pasos:** Acciones a realizar
7. **Firma:** Validacion de los participantes
