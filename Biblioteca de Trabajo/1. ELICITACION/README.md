#  1. ELICITACION - Kairos Mix

## Descripción

Esta carpeta contiene toda la documentación relacionada con el proceso de elicitación y análisis de requisitos del proyecto Kairos Mix.

---

## Contenido

### 1.0 Línea Base
Versiones de referencia aprobadas de documentos clave del proyecto.

### 1.1 Especificación de RS
Software Requirements Specification (SRS) - Documento de especificación de requisitos del sistema.

### 1.2 Cronograma
Planificación temporal del proyecto, hitos y entregas.

### 1.3 Matriz IREB
Matriz de trazabilidad de requisitos según estándares IREB.

### 1.4 Historias de Usuario
Historias de usuario en formato ágil para las funcionalidades del sistema.

### 1.5 Actas de Reunión
Registro de reuniones con stakeholders y decisiones tomadas.

### 1.6 Backlog
Product backlog priorizado con todas las funcionalidades pendientes y completadas.

### 1.7 Pruebas Unitarias
Documentación y casos de prueba unitaria.


---

##  Propósito

Documentar todos los requisitos funcionales y no funcionales del sistema Kairos Mix, incluyendo:
- Gestión de Productos
- Gestión de Clientes
- Gestión de Pedidos
- Diseño de Mezclas Personalizadas con Cálculo Nutricional
