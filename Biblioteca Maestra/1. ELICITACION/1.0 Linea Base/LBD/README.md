# Linea Base LBD - Diseño

Esta carpeta contiene todos los artefactos de diseño del sistema Kairos Mix, siguiendo el estandar IEEE Std 1016-2009.

## Contenido principal:
- Diagrama de Clases (Producto, Pedido, Cliente, MezclaPersonalizada, NutricionalService, LogObserver)
- Diagramas de Secuencia (Interaccion entre capas)
- Diagrama Entidad-Relacion (Base de Datos MySQL)
- Diagrama de Componentes (Backend Node.js y Frontend React)
- Diagramas de Estados (Ciclo de vida de Pedidos)
- Documentacion de Patrones (MVC y Observer)

## Viewpoints IEEE 1016-2009:
1. Logico (Logical)
2. Interaccion (Interaction)
3. Informacion (Information)
4. Dinamica de Estados
